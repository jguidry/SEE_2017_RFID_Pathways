Add/Remove Users


Add Users

This page will contain everything needed to register a new visitor. This functionality will be useful in the situation that the registration kiosk is down or users want their tap-cards registered to them beforehand (e.g., class field trips).


Remove Users

This page will contain everything needed to remove an existing visitor from the database. This functionality will be useful in the situation that users have not visited the aquarium in over a year.
